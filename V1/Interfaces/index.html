Interfaces

Purpose

These are the interfaces defined by this library.

Interfaces List

Interface Description
Assurance inspect a returned value, or a calculated / generated value
Check interface for true / false inspections
Inspection common standard for all assurances and requirements
ListAssurance inspect a list of returned values, or a list of calculated / generated values
ListCheck interface for true / false inspections of lists
ListInspection inspect every element of a list
ListRequirement inspect a list of input values
Requirement inspect an input value
